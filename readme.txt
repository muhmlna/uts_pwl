# instruksi menjalankan aplikasi

buka terminal dan arahkan ke folder backend
lalu jalankan perintah python init_db.py untuk membuat database
lalu jalankan perintah python server.py untuk menjalankan backend
biarkan terminal backend tetap berjalan

buka terminal lain dan arahkan ke folder frontend
lalu jalankan perintah npm install untuk menginstall dependency
lalu jalankan perintah npm start untuk menjalankan frontend
buka browser dan buka alamat http://localhost:3000 untuk melihat aplikasi
